# site-slotgames-journal

本仓库用于归档和发布多个独立 HTML 页面。内容主要涉及特定领域的静态页面，不针对任何单一域名或具体网站进行定向部署。

## 项目简介

该仓库旨在保存一系列独立的 HTML 档案，便于版本管理、历史回溯与静态部署。每个页面均为独立的可访问单元，不依赖后端服务或数据库。

## 目录说明

```
├── index.html         # 仓库首页 / 页面导航
├── pages/             # 存放各独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/            # 公共资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md          # 本文件
└── .gitignore         # 忽略规则
```

各页面文件以独立文件形式存放，不共享全局配置。资源文件统一放置于 `assets/` 目录下，便于维护。

## 页面归档说明

- 每个 HTML 文件均包含完整的前端结构，可直接在浏览器中打开。
- 页面之间无跨域或相互依赖关系。
- 文件名采用 `page-编号.html` 格式，便于按序管理。
- 如需新增页面，请遵循相同命名规则，并确保引用资源路径正确。

## 维护说明

- 本仓库仅用于归档静态 HTML 页面，不涉及动态内容生成。
- 欢迎通过 Pull Request 提交新增页面或修正现有页面。
- 提交前请确认 HTML 文件完整性及资源引用相对路径有效。
- 不建议在页面中包含外部未知来源的脚本或链接，以免影响安全性。
